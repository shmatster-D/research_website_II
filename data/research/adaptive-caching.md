Edge devices rarely have enough memory to keep every model layer warm. This project builds a caching layer that predicts which sub-networks a request will need *before* it arrives, using lightweight signals from upstream traffic.

- Cut p99 latency by **38%** on a fleet of 200 simulated edge nodes
- Built in collaboration with **Priya Natarajan** and **Tomas Berg**
- Funded by the Linden University Systems Lab seed grant
